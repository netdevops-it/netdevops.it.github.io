# 📊 Intro to Source of Truth

Introduction to Source of Truth platforms: NetBox, Nautobot, and OpsMill. Learn how to establish a single source of truth for your network infrastructure.

## Course Overview

This one-day intensive course introduces you to the concept of Network Source of Truth (SSoT) and provides hands-on experience with the leading platforms: NetBox, Nautobot, and OpsMill. Learn how to establish a single source of truth for your network infrastructure.

## Course Details

- **Duration:** 1 day
- **Price:** Contact us for pricing
- **Prerequisites:** None required
- **Format:** In-person or virtual training available

## What You'll Learn

### Source of Truth Concepts
- Understanding the importance of SSoT in network automation
- Data modeling and relationships
- IPAM (IP Address Management) fundamentals
- DCIM (Data Center Infrastructure Management) concepts
- Integration with automation tools

### Platform Overviews

#### NetBox
- Installation and configuration
- Data modeling and custom fields
- IPAM and device management
- API usage and automation integration

#### Nautobot
- Extensible platform architecture
- Custom fields and relationships
- Jobs and automation workflows
- GraphQL and REST API
- Plugin ecosystem

#### OpsMill
- Modern platform features
- Workflow automation
- Integration capabilities
- Use cases and best practices

### Hands-On Exercises
- Setting up a basic SSoT instance
- Data import and migration strategies
- API integration examples
- Automation workflows
- Best practices and recommendations

## Course Benefits

- **Platform Comparison:** Understand the differences between leading SSoT platforms
- **Practical Experience:** Hands-on labs with each platform
- **Best Practices:** Learn industry-standard approaches
- **Immediate Application:** Apply concepts to your environment right away
- **Expert Guidance:** Learn from experienced practitioners

## Who Should Attend

- Network engineers implementing automation
- Network administrators managing infrastructure
- DevOps engineers working with network infrastructure
- IT managers evaluating SSoT solutions
- Anyone interested in network automation foundations

## Prerequisites

- Familiarity with web interfaces
