# 📊 Introductie Source of Truth

Introductie tot Source of Truth platforms: NetBox, Nautobot en OpsMill. Leer hoe je een single source of truth opzet voor je netwerk infrastructuur.

## Cursus Overzicht

Deze intensieve eendaagse cursus introduceert je tot het concept van Network Source of Truth (SSoT) en biedt hands-on ervaring met de leidende platforms: NetBox, Nautobot en OpsMill. Leer hoe je een single source of truth opzet voor je netwerk infrastructuur.

## Cursus Details

- **Duur:** 1 dag
- **Prijs:** Neem contact op voor prijzen
- **Voorkennis:** Geen vereist
- **Formaat:** Op locatie of virtuele training beschikbaar

## Wat Je Leert

### Source of Truth Concepten
- Begrijpen van het belang van SSoT in netwerk automatisering
- Data modeling en relaties
- IPAM (IP Address Management) fundamentals
- DCIM (Data Center Infrastructure Management) concepten
- Integratie met automatisering tools

### Platform Overzichten

#### NetBox
- Installatie en configuratie
- Data modeling en custom fields
- IPAM en device management
- API gebruik en automatisering integratie

#### Nautobot
- Uitbreidbaar platform architectuur
- Custom fields en relaties
- Jobs en automatisering workflows
- GraphQL en REST API
- Plugin ecosysteem

#### OpsMill
- Moderne platform features
- Workflow automatisering
- Integratie mogelijkheden
- Use cases en best practices

## Voor Wie

- Netwerk engineers die automatisering implementeren
- Netwerk administrators die infrastructuur beheren
- DevOps engineers die werken met netwerk infrastructuur
- IT managers die SSoT oplossingen evalueren
- Iedereen geïnteresseerd in netwerk automatisering fundamenten
